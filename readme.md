# Two Bar Linkage Kinematics and Path Planning

The repository contains c++ implementation of a two bar linkage with 2 DOF. 

To build:

```sh
    mkdir build
    cd build && cmake ..
    make
```

To run the example:
```sh
    cd build
    ./TwoBarLinkage
```
